### Integration test (qa/be)

```
tests/e2e/sprint3/ics-export.spec.ts
```

```typescript
test('GET /api/users/me/shifts/export — scarica file .ics valido', async ({ employeePage }) => {
  // Setup: dipendente con 2 turni nel range
  const [download] = await Promise.all([
    employeePage.waitForEvent('download'),
    employeePage.click('[data-testid="export-ics-btn"]'),
  ]);
  expect(download.suggestedFilename()).toMatch(/turni.*\.ics$/);
  const content = await (await download.createReadStream()).read();
  expect(content.toString()).toContain('BEGIN:VCALENDAR');
  expect(content.toString()).toContain('BEGIN:VEVENT');
});

test('GET /api/users/me/shifts/export — non espone turni di altri (T-SEC-01)', async ({ employeePage, otherUserShiftId }) => {
  // L'export filtra SOLO per session.user.id
  const resp = await employeePage.request.get('/api/users/me/shifts/export');
  const icsText = await resp.text();
  expect(icsText).not.toContain(otherUserShiftId);
});

test('GET /api/users/me/shifts/export — utente non autenticato → 401', async ({ page }) => {
  const resp = await page.request.get('/api/users/me/shifts/export');
  expect(resp.status()).toBe(401);
});
```

## Acceptance Criteria

- [ ] `ics` presente in `package.json` come dipendenza diretta (non solo `@types/ics`)
- [ ] `GET /api/users/me/shifts/export?from=2026-07-01&to=2026-07-31` con utente autenticato e turni nel periodo → risposta 200, `Content-Type: text/calendar`, body inizia con `BEGIN:VCALENDAR`
- [ ] File scaricato contiene `BEGIN:VEVENT` + `DTSTART` + `SUMMARY` per ogni turno nel range
- [ ] Range vuoto (nessun turno nel periodo) → risposta 200 con calendario valido ma 0 VEVENT (non un errore)
- [ ] T-SEC-01: il file non contiene turni di altri utenti anche se passato `?userId=altro`
- [ ] Utente non autenticato → 401
- [ ] Filename header: `Content-Disposition: attachment; filename="turni_2026-07-01_2026-07-31.ics"` quando from/to specificati
- [ ] CalendarToolbar: il pulsante "Esporta .ics" è visibile e scarica il file per il periodo visualizzato
