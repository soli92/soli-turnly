# TSK-014 — Dashboard Admin page

## Obiettivo

Implementare la schermata di sintesi operativa dell'admin (RF-K): KPI cards con dati
reali, inbox richieste con contatore live, e scorciatoie alle azioni frequenti.

## Componenti da creare

```
app/(admin)/dashboard/page.tsx          ← RSC: fetch parallelo dei KPI
components/dashboard/
  KpiCard.tsx                           ← Card generica (titolo, valore, variazione, icona)
  CoverageAlertList.tsx                 ← Lista fasce sotto-coperte (RB-07)
  ViolationSummary.tsx                  ← Contatore violazioni aperte
  InboxBadge.tsx                        ← Badge richieste in attesa (link a /admin/requests)
  QuickActionsBar.tsx                   ← Scorciatoie: Nuova assegnazione, Nuova assenza
  OvertimeSummaryCard.tsx               ← Ore straordinario del periodo
```

### KPI da mostrare

| KPI | Fonte API | Aggiornamento |
|---|---|---|
| Richieste in attesa | `GET /api/requests?status=sent&limit=1` (count) | TanStack Query, refetchInterval 60s |
| Fasce sotto-coperte oggi | calcolo client da `GET /api/admin/coverage` + `GET /api/shifts` | on mount |
| Violazioni aperte | `GET /api/shifts?hasViolations=true&from=today` (count) | on mount |
| Ore straordinario settimana | `GET /api/admin/reports/overtime?period=current_week` | on mount |
| Assenze in corso | `GET /api/admin/absences?status=approved&activeToday=true` (count) | on mount |

### Skeleton loading

- Timeout > 5s: mostra stato errore con "Riprova"

## Criteri di accettazione

- [ ] I 5 KPI mostrano valori coerenti con il DB (RF-K CA1 — nessun valore statico)
- [ ] InboxBadge: dopo invio richiesta da dipendente, il contatore si aggiorna entro 30s
  senza refresh (RF-K CA2, T-INT-03)
- [ ] CoverageAlertList: se 0 sotto-coperture → empty state "Copertura OK"
- [ ] QuickActionsBar: link a `/admin/matrix` e `/admin/absences` navigano correttamente
- [ ] Skeleton loader visibile durante fetch; errore gestito con messaggio + retry
- [ ] Layout responsive: cards si dispongono in 2 colonne su mobile, 3 su desktop
- [ ] Navigazione tastiera: ogni card/link raggiungibile con Tab
