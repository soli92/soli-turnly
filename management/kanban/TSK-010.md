# TSK-010 — Test di accettazione Playwright

## Obiettivo
Implementare la suite Playwright che copre i test cases definiti in
`code_quality/acceptance/turnly-acceptance.yaml`, coprendo i flussi critici
con fixture, helper e setup deterministico.

## Setup Playwright

```typescript
// playwright.config.ts
export default defineConfig({
  testDir: './tests/e2e',
  fullyParallel: true,
  use: {
    baseURL: process.env.PLAYWRIGHT_BASE_URL ?? 'http://localhost:3000',
    trace: 'on-first-retry',
  },
  projects: [
    { name: 'chromium', use: { ...devices['Desktop Chrome'] } },
    { name: 'firefox',  use: { ...devices['Desktop Firefox'] } },
  ],
  webServer: {
    command: 'npm run start:test',
    url: 'http://localhost:3000',
    reuseExistingServer: !process.env.CI,
  },
});
```

## Test cases da implementare

### T-DOM (Dominio turni)
```typescript
// tests/e2e/domain/shifts.spec.ts

test('T-DOM-02: RB-01 blocca sovrapposizione', async ({ adminPage }) => {
  await adminPage.goto('/admin/matrix');
  // Assegna turno M 08:00-16:00 a Mario Rossi Lunedì
  // Poi tenta assegnare turno M2 10:00-18:00 stesso giorno
  // Assert: ShiftEditor mostra errore RB-01, bottone Salva disabilitato
  await expect(adminPage.getByTestId('violation-badge-RB-01')).toBeVisible();
  await expect(adminPage.getByRole('button', { name: 'Salva' })).toBeDisabled();
});

test('T-DOM-08: RB-12 DST — durata corretta ora solare/legale', async ({ adminPage }) => {
  // Turno notturno dal 30 Mar 2025 01:00 → 31 Mar 2025 08:00 (DST forward)
  // Durata attesa: 6h (non 7h)
  await adminPage.goto('/admin/matrix?date=2025-03-30');
  await expect(adminPage.getByTestId('shift-duration')).toHaveText('6h');
});
```

### T-REC (Ricorrenze)
```typescript
// tests/e2e/domain/recurrence.spec.ts

test('T-REC-01: ricorrenza salta assenza approvata', async ({ adminPage }) => {
  // Pre: assenza approvata 15-17 gennaio per Lucia Verdi
  // Genera ricorrenza settimanale 8-31 gennaio
  // Assert: nessun turno creato il 15, 16, 17
  await adminPage.goto('/admin/recurrence');
  // ... wizard steps
  const shifts = await adminPage.getByTestId('recurrence-preview-shifts').allTextContents();
  expect(shifts).not.toContain('15 gen');
});
```

### T-REQ (Richieste dipendente)
```typescript
// tests/e2e/domain/requests.spec.ts

test('T-REQ-01: flusso approvazione completo', async ({ adminPage, employeePage }) => {
  // Dipendente invia richiesta assenza
  await employeePage.goto('/requests/new');
  await employeePage.getByRole('radio', { name: 'Assenza' }).click();
  await employeePage.getByLabel('Data inizio').fill('2025-02-10');
  await employeePage.getByLabel('Data fine').fill('2025-02-12');
  await employeePage.getByRole('button', { name: 'Invia richiesta' }).click();

  // Admin approva
  await adminPage.goto('/admin/requests');
  await adminPage.getByTestId('request-item-1').click();
  await adminPage.getByRole('button', { name: 'Approva' }).click();

  // Dipendente vede approvazione
  await employeePage.goto('/requests');
  await expect(employeePage.getByText('Approvata')).toBeVisible();
});
```

## Criteri di accettazione

- [ ] Setup auth con `storageState` (nessun login UI nei test di dominio)
- [ ] Tempo totale suite < 5 min su runner GitHub Actions standard
- [ ] Report HTML generato in `playwright-report/` con screenshot su failure
