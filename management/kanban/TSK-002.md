# TSK-002 — Schema database

## Obiettivo
Definire lo schema Drizzle completo (10 tabelle), generare le migration SQL,
e creare un seed script per i dati di test.

## Schema (db/schema.ts)

```typescript
import { pgTable, pgEnum, uuid, text, integer, boolean,
         timestamp, jsonb, index, unique } from 'drizzle-orm/pg-core';
import { sql } from 'drizzle-orm';

// Enums
export const userRoleEnum = pgEnum('user_role', ['admin', 'employee']);
export const shiftStatusEnum = pgEnum('shift_status', ['planned', 'confirmed', 'cancelled']);
export const absenceTypeEnum = pgEnum('absence_type', ['vacation', 'sick', 'leave']);
export const requestTypeEnum = pgEnum('request_type',
  ['absence', 'shift_swap', 'new_shift', 'modify_shift']);
export const requestStatusEnum = pgEnum('request_status',
  ['draft', 'sent', 'awaiting_colleague', 'approved', 'rejected', 'cancelled', 'applied']);

// Tables (10)
export const qualifications = pgTable('qualifications', { ... });
export const users = pgTable('users', { ... });
export const shiftTypes = pgTable('shift_types', { ... });
export const shifts = pgTable('shifts', { ... },
  (t) => ({
    // CRITICAL: RB-01 overlap prevention + T-INT-02
    overlapExclude: sql`EXCLUDE USING gist
      (${t.userId} WITH =, tsrange(${t.startDt}::timestamp, ${t.endDt}::timestamp) WITH &&)`
  })
);
export const recurrenceRules = pgTable('recurrence_rules', { ... });
export const absences = pgTable('absences', { ... });
export const availability = pgTable('availability', { ... });
export const requests = pgTable('requests', { ... });
export const coverageRequirements = pgTable('coverage_requirements', { ... });
export const swapOperations = pgTable('swap_operations', { ... });
export const notifications = pgTable('notifications', { ... });
export const auditLog = pgTable('audit_log', { ... });
```

## Indici critici
- `shifts(user_id, start_dt, end_dt)` — EXCLUDE USING gist (RB-01, T-INT-02)
- `notifications(recipient_id, read)` — inbox counter O(1)
- `requests(user_id, status)` — lista richieste dipendente e coda admin
