# TSK-005 — Matrice turni admin

## Obiettivo
Implementare la schermata centrale dell'applicazione: griglia dipendenti × giorni
con TanStack Table v8 virtualizzato, validazione inline, e modal editor turno.

## Componenti da creare

### ShiftGrid (RSC + Client boundary)
- `components/matrix/ShiftGrid.tsx` — TanStack Table v8
- `components/matrix/ShiftCell.tsx` — cella singola (turno/vuoto/assenza)
- `components/matrix/ViolationBadge.tsx` — badge ⚠ / ✕ inline
- `components/matrix/ShiftEditor.tsx` — modal editor (shadcn Dialog)
- `components/matrix/MatrixFilters.tsx` — filtri qualifica + search dipendente
- `components/matrix/WeekNavigator.tsx` — frecce settimana/mese

### Logica TanStack Table
```typescript
const table = useReactTable({
  data: employees,             // 50+ righe
  columns: dayColumns,         // 7 o 31 colonne
  getCoreRowModel: getCoreRowModel(),
  // Column pinning: nomi dipendenti sticky a sinistra
  state: { columnPinning: { left: ['employee'] } },
});
```

### Virtualizzazione
- `@tanstack/react-virtual` per righe (50+ employees)
- Colonne fisse (7 o 31): nessuna virtualizzazione orizzontale necessaria

### Validazione inline
- Al drop/click su cella: chiama `lib/rules/` (TSK-006) con stato corrente
- Errori bloccanti: bordo rosso + tooltip RB-xx, salvataggio bloccato
- Avvisi: bordo ambra + tooltip, salvataggio con conferma (shadcn AlertDialog)

### Vista settimana vs mese
- Toggle in header: "Vista settimana" / "Vista mese"
- Settimana: celle ampie (120px min) con nome tipologia + orari
- Mese: celle compatte (36px) con solo lettera iniziale + colore

## Criteri di accettazione
- [ ] Griglia 50 dipendenti × 31 giorni carica in < 2s (T-RNF-01)
- [ ] Scroll fluido orizzontale e verticale senza blocchi
- [ ] Header dipendenti sticky a sinistra, header giorni sticky in alto
- [ ] Click su cella apre ShiftEditor modal con dati pre-compilati
- [ ] Violazione RB-01 (sovrapposizione) mostra bordo rosso + salvataggio bloccato — T-DOM-02
- [ ] Avviso RB-02 (riposo) mostra bordo ambra + conferma richiesta — T-DOM-03
- [ ] Celle di assenza non cliccabili/assegnabili — T-DOM-04
- [ ] Filtro qualifica: mostra solo righe con qualifica selezionata
- [ ] Vista mese: celle 36px con padding sufficiente, overflow gestito
